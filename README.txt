[개발 도구]

- Visual Studio Code : 코드 작성 (구름 ide에서는 selenium 설치 이슈가 있어 부득이하게 vscode 사용)
- ngrok: Ngrok를 사용하여 로컬에서 실행 중인 Quart 서버를 공개 인터넷으로 노출해주는 배포 도구
  - ngrok 가입 및 다운로드: https://ngrok.com/	

[라이브러리 및 프레임워크]

- Quart: Quart를 활용하여 비동기 웹 어플리케이션을 개발
- Asyncio: 비동기 프로그래밍을 위해 asyncio 라이브러리 활용
- asyncgpt: GPT 모델과 비동기적 상호 작용을 위한 asyncgpt 라이브러리 사용
- 기타 라이브러리: 시스템 관련 기능을 위해 sys와 같은 라이브러리 활용, 크롤링을 위해 selenium 라이브러리 활용
